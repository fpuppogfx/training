Just do it

Create a new file

  • Login to Figma
  • Click on New Design File
  • Give it a name

Set up a frame

  • Use the Frame tool (F) to create your first layer
  • Pick a frame size from the ones available
  • Don’t worry, you can always edit it later

Design a layout

  • Use the toolbar to create shapes and add text1
  • Use the right sidebar to edit and customise elements

  1. Draw rectangles (R), circles and ovals (O), and add text (T) ↩︎

Clean it up

  • Use the right sidebar to navigate through layers and rename them
  • Move layers or use the right click menu to define the order of the layers

Import images

  • Drag and drop a file in Figma to import and image
  • Or apply it as a background to a shape
  • Inspect an element and use the right sidebar to replace it

Import SVGs

  • Same as for images, drag and drop an SVG in Figma
  • Inspect it to edit it’s colour and change the line weight

Import fonts

  • Desktop app: Figma will automatically read the files installed in your computer1
  • Web app: follow the instructions in this page to install the font service

  1. If this doesn’t work on a newly installed font, try restarting your machine ↩︎

Export it

  • Select the frame and use the right sidebar to export it in the formats you prefer
  • You can export multiple formats at once
  • Folders will be automatically created based on the frame name1

  1. Figma reads slashes (/) as new folders ↩︎

Do it properly

Load a library

  • In left sidebar, select Assets, then the book icon, and load an existing library
  • Feel free to use any library you want since, while in your file, you will not edit anything in the source library

Use styles and components from the library

  • Click on one of the elements you created before and load a style from the library
  • Navigate through the assets in the left sidebar and drag and drop them in your file

But what if I want to create my own library?

Set up global colours

  • Select an element and navigate to the property you want to save (i.e. it’s colour)
  • Click on the +, give it a name and voilà
  • Apply the newly created global style to another item

Set up global text styles

  • Select a text element
  • In the right sidebar’s lick on the icon next to Text and then create a new font style
  • Apply the newly created global style to another item

Edit global styles

  • Click on an empty space in your Figma file
  • On the right sidebar you can see a list of all the styles belonging to this file
  • Edit them from here to automatically apply them to every item that uses them

Create components

  • Select an element you want to reuse
  • Click on Create component (Shift + CTRL + K)1

  1. Better also rename it to something easy to understand ↩︎

Editing components

  • Import the newly created component from the Assets library
  • Go back to the main component and change something
  • Check that the changes have been duplicated in the imported instance

Breaking components

  • Right click on an instance and select Detach instance (Alt + CTRL + B)
  • Any change to the main component will not impact this element
  • We try to use this as less as possible since it makes it difficult to maintain consistency

Publish your library

  • In the left sidebar, under Assets, click on the book icon and the on Publish to allow everyone to load components and global styles in another file

Do it better

Files and Folders

  • Use the menu in the toolbar to move the file to a project
  • If a file is not in a project, Figma will put it in the Drafts
  • Drafts are private unless manually shared

Pages and Sections

  • In the left sidebar, create a new page to organise your file
  • You can create sections (Shift + S) to group items in your pages

Sharing

  • In the toolbar, click on Share to allow others to view or edit your file
  • You can also share a link that navigates the user to the currently select item

Collaboration

  • Add comments using the button in the toolbar or by pressing C
  • Comments will stay attached to the element they have been added to even if you move it
  • You can tag people using @ and mark comments as resolved once they are not no longer needed1

  1. Users need to have an account in Figma and will receive a notification via email ↩︎

Prototypes

  • Using the right sidebar, switch to Prototype mode
  • Create connections between elements to link them
  • Use the popup to modify animation settings
  • Click on the Present icon in the toolbar to view your prototype1

  1. Prototypes can be shared separately from the Figma file ↩︎
